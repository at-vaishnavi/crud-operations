# Library CRUD Operations (Python + SQLite)

## Project Overview
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Python and SQLite database.

It is a simple console-based application for managing library records.

---

## Features
- Add new book records
- View existing books
- Update book details
- Delete book records

---

## Technologies Used
- Python
- SQLite

---

## How to Run

1. Install Python.
2. Download or clone this repository.
3. Open terminal in the project folder.
4. Run
